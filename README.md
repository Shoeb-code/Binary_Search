## 🚀 What is Binary Search?

**Binary Search** is a searching algorithm used on **sorted arrays or search spaces**. It repeatedly divides the search interval in half, reducing the search space efficiently.

### Time Complexity

| Case    | Complexity   |
| ------- | ------------ |
| Best    | **O(1)**     |
| Average | **O(log n)** |
| Worst   | **O(log n)** |

### Space Complexity

* **Iterative:** O(1)
* **Recursive:** O(log n)
